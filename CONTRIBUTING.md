# Contributing to Coldrun

## Repository Scope

This repository is for product code and related engineering artifacts for Coldrun.

## AI Artifact Separation Rule

Do not store AI-related files in this repository.

AI-related artifacts include:

- AI planning notes
- AI discussions and brainstorming
- AI prompt/session artifacts
- AI-generated design exploration documents

Store all AI artifacts in the sibling repository: ../Coldrun-AI

## Notes

- Keep documentation in this repository focused on product behavior, implementation, and operations.
- If in doubt, place exploratory AI material in Coldrun-AI and keep finalized technical decisions in product docs.
